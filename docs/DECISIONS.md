# 决策记录

## 2026-07-15 产品与技术基线

- 应用名称为“易读 / Easy Book”，界面使用简体中文。
- 首发为个人使用的 Android APK，使用 PakePlus 包装 React 静态产物。
- 前端采用 Vite、React、TypeScript；路由采用 Hash Router。
- 小说及阅读数据使用 Dexie/IndexedDB，不使用服务器。
- 首期只导入 TXT；自动识别章节，不提供元数据或章节人工修正。
- MVP 采用“书架 + 设置”两栏；详情和阅读器按层级进入。
- 阅读方式为纵向滚动，支持多阅读主题、字号、行距和亮度遮罩。
- 首期包含目录跳转、书签和书内全文搜索。
- 首期不做备份恢复；卸载或清除应用数据会丢失书库。
- EPUB、TTS、账号、同步、内容分发及商业化明确排除。

## 设计裁决

- `stitch_/` 永久作为只读参考，不迁入其 CDN、远程图片或演示业务。
- 颜色与语义以原 `DESIGN.md` 为基础；基础背景统一为 `#131313`。
- 卡片圆角采用设计文档的 16px，不采用导出 HTML 的 8px 配置。
- 中文字体需本地化；Newsreader、Literata 和 Inter 的角色关系保留，但不能依赖 Google Fonts。
- 原五栏导航缩减为“书架 + 设置”；“我的”与商业化入口不进入 MVP。
- 阅读器设置抽屉默认收起，符合沉浸式阅读目标。

## 默认实现值

- 包名：`com.easybook.reader`
- 包管理器：pnpm
- TXT 上限：50 MiB
- 解码：BOM → 严格 UTF-8 → GB18030
- 未识别章节：整本作为“正文”
- 默认字号：19px；默认行高：1.8；默认主题：charcoal
- 进度写入节流：500ms
- 搜索结果设置上限并允许取消

## PakePlus 风险

PakePlus 支持静态 React `dist` 打包 Android，但其公开文档未充分定义 WebView 中 IndexedDB、文件选择器和版本更新行为。正式功能依赖以下真机验证：

- Android 文件选择器能返回 TXT 内容。
- Hash 路由冷启动与返回行为正确。
- IndexedDB 在进程结束、设备重启后保留。
- 同包名覆盖安装新版本后数据库保留。
- 安全区、软键盘和状态栏表现正确。

若文件或数据库任一关键能力不可靠，不以 localStorage 或 Base64 嵌入正文规避；暂停并评估支持原生文件与 SQLite 的替代壳。
