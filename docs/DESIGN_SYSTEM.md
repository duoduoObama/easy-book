# Easy Book 设计系统

设计方向继承 `stitch_/lumina_novel/DESIGN.md` 的 “Immersive Sanctuary”：安静、克制、编辑感，优先服务长时间阅读。本文是实现时的最终规范。

## 设计令牌

### 应用暗色界面

- `--surface: #131313`
- `--surface-low: #1c1b1b`
- `--surface-container: #201f1f`
- `--surface-high: #2a2a2a`
- `--surface-highest: #353534`
- `--text: #e5e2e1`
- `--text-muted: #bfaea5`（由原稿次要文字降低暖度以改善层级）
- `--primary: #ffb689`
- `--primary-strong: #f08d49`
- `--outline: #554339`
- `--error: #ffb4ab`

以 `#131313` 为统一背景，解决原稿 `#121212` 与 `#131313` 冲突。深度依靠色阶和低对比 1px 描边，不使用大面积重阴影。

### 阅读主题

| 主题 | 背景 | 正文 |
|---|---|---|
| charcoal | `#131313` | `#dbcfc9` |
| paper | `#f4ecd8` | `#40382f` |
| sage | `#e1edd8` | `#344032` |
| mist | `#e6f3f4` | `#304144` |
| black | `#090909` | `#d6d2cf` |
| graphite | `#2c2c2c` | `#dedad7` |

亮度通过固定黑色遮罩实现，不调用设备系统亮度。

## 字体

字体必须本地打包。界面字体优先使用随包的中文无衬线字体并回退系统 sans-serif；标题和正文使用随包的中文阅读字体并回退 serif。原稿 Newsreader/Literata 仅覆盖拉丁字符，不能单独承担中文正文。

- 页面标题：32/40，600
- 章节标题：24/32，600
- 卡片标题：18/26，600
- 阅读正文：默认 19px，行高 1.8，可调 16–28px 与 1.5–2.2
- 界面正文：14/22
- 辅助标签：12/18，500

正文按段落留白，不强制首行缩进；保留 TXT 中的有效空行。

## 间距与形状

- 页面水平边距：20px；阅读器 24px
- 页面顶部内容间距：24–32px
- 栅格间距：16px
- 书架封面比例：3:4
- 阅读列最大宽度：700px
- 小控件圆角：8px；卡片：16px；抽屉：24px；胶囊：999px
- 书籍封面圆角：8px

本文采用设计文档的 16px 卡片圆角，覆盖导出 HTML 中较小的重复配置。

## 组件

- 主按钮：暖橙实心、深色文字、最小高度 48px。
- 次按钮：透明背景、低对比描边。
- 书籍卡：自动生成有辨识度的渐变封面，显示进度；不使用网络图片。
- 章节项：最小高度 56px，当前章节使用左侧 primary 标记。
- 底部导航：仅“书架”“设置”，安全区内最小高度 64px。
- 阅读控制：默认收起；中央轻触显示，滚动向下隐藏。
- 搜索：圆角输入，结果必须显示章节名和命中上下文。

## 动效

- 页面进入 180–240ms；抽屉 280ms；点击反馈 100ms。
- 不使用连续装饰动画。
- `prefers-reduced-motion: reduce` 时禁用位移和渐入。

## 无障碍与移动端

- 点击目标至少 44×44 CSS px。
- 正文允许选择；图标按钮有 `aria-label`。
- 焦点可见，颜色不是唯一状态提示。
- 使用 `viewport-fit=cover` 与 `env(safe-area-inset-*)`。
- 支持 320px 宽屏幕，不产生水平滚动。
- 深色文本与背景、亮色主题文本与背景均满足至少 WCAG AA。
